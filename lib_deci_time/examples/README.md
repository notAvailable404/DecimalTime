# Decimal Time Examples

These scripts demonstrate the `deci_time` library.

## Prerequisites
- Python 3.7+
- The `deci_time.py` library must be in the parent directory (or installed in site-packages).

## How to run
From the terminal, inside this `examples` folder:

```bash
python3 ex01_basic_usage.py
python3 ex02_math_and_conversion.py
python3 ex03_custom_planets.py
